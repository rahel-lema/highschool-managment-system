# High School Management System

A web-based High School Management System designed to simplify and manage daily school activities.
The system provides separate dashboards for administrators, teachers, and students, making school manag
ement more efficient and organized.

Here are the Features found in this project
👨‍💼 Admin
- Secure login
- Manage students
- Manage teachers
- Manage classes
- Manage subjects
- Post announcements
- View reports

Teacher
- Secure login
- View assigned classes
- Upload assignments
- Enter student grades
- Post announcements
👨‍🎓 Student
- Secure login
- View profile
- View grades
- View assignments
- Read announcements
🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP (Apache & MySQL)

## 🚀 Installation

1. Clone the repository:
  
   git clone https://github.com/yourusername/high-school-management-system.git
   
2. Move the project to the XAMPP htdocs folder.

3. Start Apache and MySQL from XAMPP.

4. Import the SQL database into phpMyAdmin.

5. Open your browser and visit:

  
   http://localhost/high-school-management-system/
